# RSS Feed Project

This project fetches and processes RSS feeds, logs the results, and saves new items into separate XML files.

## Project Structure

- `data_lake/`: Directory to store the fetched RSS feed XML files.
- `rss_urls.xlsx`: Spreadsheet containing the list of RSS feed URLs.
- `feed_logs.xlsx`: Spreadsheet for logging the feed processing runs.
- `rss_reader.py`: Script to fetch, process, and log RSS feeds.
- `README.md`: Project documentation.
